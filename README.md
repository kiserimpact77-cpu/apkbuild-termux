# apkbuild-termux
Build Android APK from your phone - No PC required!

## Requirements
- Termux
- proot-distro Ubuntu
- Android SDK 33
- Java 17
- Gradle 7.5

## Installation
bash setup.sh

## Usage
apkbuild new myapp
apkbuild build myapp

## Notes
- First setup takes 30 minutes
- APK saved to Downloads
- No root required
- Tested on ARM64
